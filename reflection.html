reflection

  • Responsive Design is a kind of web design allow website layout and appearance that can be flexibly adapted to fit all device screens width such as desktops, tablets, smartphones at all resolution levels.
  • Responsive Design has changed the way programmer design "multi-device web" create a layout that can respond to the device being used to view the content.
  • Nowadays, The advent and development of electronic devices with different screen's widths and resolutions has led to the development of Web technology. Web Responsive, which focus on Responsive Design, is a trend and developed to allow users to access a website on any device. Due to only one website so it will reduce costs, easily maintain and develop website.
  • In this assignment, I used “@media” technique that Media queries allow websites to use different CSS style rules based on the characteristics of the device on which the website is displayed and setting the page element size to the corresponding unit percentage, instead of the absolute percentage unit such as pixels or points
  • I learned what Responsive Design is and how to make a multi-device web base on Responsive Design. Clearly, building web apps for diferent devices is make the site more friendly to all devices
